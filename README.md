# Generative AI for Data Science: Lab Solutions

## 📌 Project Overview
This repository contains my solutions for a practical lab focused on utilizing Generative AI and prompt engineering to solve real-world data science and analytics problems. The project demonstrates how to iteratively guide AI models to generate, debug, and optimize Python code and SQL queries through structured prompting.

## 🛠️ Scenarios Covered

### 1. Retail Inventory Analysis
* **Goal:** Analyze inventory turnover, identify slow-moving items, and predict potential stockouts.
* **Tech Stack:** Python, Pandas, Matplotlib, Seaborn.
* **AI Focus:** Prompted the AI to act as a Senior Supply Chain Data Scientist. Used follow-up prompts to implement robust data validation, handle zero-division errors natively, and optimize visualizations for stockout risks by category.
